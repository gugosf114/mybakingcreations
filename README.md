A complete single-page website built for **My Baking Creations**, a family-owned San Francisco bakery.

## Project Overview

**Client:** My Baking Creations (family business)  
**Timeline:** 1 session  
**Role:** Full build — design implementation, responsive layout, form integration

## Features

- ✅ Responsive single-page design
- ✅ Brand-aligned color scheme (from existing style guide)
- ✅ Hero section with CTA buttons
- ✅ Services section with pricing
- ✅ Image gallery grid
- ✅ Corporate clients section
- ✅ Order request form (Formspree-ready)
- ✅ Contact information
- ✅ Mobile-optimized navigation
- ✅ Smooth scroll animations

## Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, Grid, Flexbox, animations
- **JavaScript** — Smooth scroll, form handling
- **Google Fonts** — Fredoka One, Nunito
- **Formspree** — Form backend (plug-and-play)

## Brand Assets Integrated

- Custom logo (icon + wordmark)
- Brand color palette: `#EC268F` (pink), `#FFC532` (yellow), `#622D2B` (dark brown)
- Product photography
- Branded packaging mockups

My Baking Creations is a family-owned bakery founded in 2012, serving custom cakes, logo cookies, and cake pops to both individual clients and Fortune 500 companies including Google, Meta, Salesforce, and DocuSign.
